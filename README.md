# Mount Zion College of Nursing

This project uses a React/Vite frontend. To make it easy to view the site without building, we use a development server.

## Quick Start
1. Double-click **`start-site.bat`** in this folder.
2. The site will automatically open in your browser at `http://localhost:5173`.
3. Any changes you make to the code in the `frontend/` folder will appear instantly!

## Opening via HTML
You can also open the `index.html` file in this folder. It will automatically try to redirect you to the "real site" running on the development server.

---

## Technical Details

### Frontend (React)
- Located in `frontend/`
- Run `npm install` inside `frontend/` if you are setting up for the first time manually.
- Run `npm run dev` to start the server.

### Backend (Node.js)
- Located in `backend/`
- Run `npm start` to start the API server (if needed for forms/data).

### Legacy Files
- The `legacy_static/` folder contains a backup of the original HTML site for reference.
